# Asia Cup 2025 Power BI Dashboard

## 📊 Project Description
This project analyzes Asia Cup 2025 data using Power BI. It provides insights into team performance, player statistics, and match results through interactive visualizations.

## 🛠 Tools Used
- Power BI
- Excel / CSV Dataset

## 📁 Project Files
- Power BI file (.pbix)
- Dashboard exported PDF
- Dataset (CSV/Excel)
- Screenshots of dashboard

## 📌 Key Features
- Team-wise performance analysis  
- Player statistics visualization  
- Match results overview  
- Interactive filters and KPIs  
- Data-driven insights  

## 🎯 Objective
To analyze cricket data of Asia Cup 2025 and present meaningful insights using visual dashboards.
